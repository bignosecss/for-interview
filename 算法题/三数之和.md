# 基本思路

保持与两数之和一致。
- 三层循环，找出所有的三数之和，一一比较
- 要注意去重

```javascript
function threeSum(nums) {
  const result = [];
  const len = nums.length;

  for (let i = 0; i < len; i++) {
    const x = nums[i];
    for (let j = i + 1; j < len; j++) {
      const y = nums[j];
      for (let k = j + 1; k < len; k++) {
        const z = nums[k];
        if (x + y + z === 0) {
          const triplet = [x, y, z].sort((a, b) => a - b);
          const found = result.find(subArr => JSON.stringify(subArr) === JSON.stringify(triplet));
          if (!found) result.push(triplet);
        }
      }
    }
  }
}
```

# 双指针优化

1. 基本方程是这样的：x + y + z = 0，可以优化成 `target = -x` -> `y + z = target`；
2. 将输入数组排好序，只需要在 x(i) 的右侧，使用双指针算法，逐渐缩小范围即可，小了 y(j++)，大了 z(k--)
3. 等于呢，也要注意去重

> 鄙人突然发现，用 `JSON.stringify()` 去重，力扣过不了测试，超时啦！

> 换一种方式，使用 `Set` 去重，过啦～

```javascript
function threeSum(nums) {
  nums.sort((a, b) => a - b);
  const len = nums.length;
  const result = [];
  const seen = new Set();

  for (let i = 0; i < len; i++) {
    const x = nums[i];
    const target = -x;

    let j = i + 1;
    let k = len - 1;
    while (j < k) {
      const y = nums[j];
      const z = nums[k];
      if (y + z < target) {
        j++;
      } else if (y + z > target) {
        k--;
      } else {
        const triplet = [x, y, z];
        const key = triplet.join(',')l
        if (!seen.has(key)) {
          seen.add(key);
          result.push(triplet);
        }
        j++;
        k--;
      }
    }
  }
}
```
