思路很简单：
1. 构建一个 hash map，存储 字符：频率
2. 遍历字符串，找到第一个频率为1的字符，返回其索引
```javascript
function firstUniqueChar(s) {
  const freq = new Map();
  for (const c of s) {
    freq.set(c, (freq.get(c) ?? 0) + 1);
  }

  for (let i = 0; i < s.length; i++) {
    const c = s[i];
    if (freq.get(c) === 1) return i;
  }

  return -1;
}
```
