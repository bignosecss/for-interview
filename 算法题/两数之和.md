# 基本思路

找出所有的两数之和，一一和 target 比较。
1. 选定一个基数，遍历数组中除基数之外的元素
2. 比较这两个数与 target 是否相等，是 true，否 false

```javascript
function twoSum(nums, target) {
  for (let i = 0; i < nums.length; i++) {
    const x = nums[i];
    // i+1 是因为之前的已经加过了，没必要重复计算
    for (let j = i + 1; j < nums.length; j++) {
      const y = nums[j];
      if (x + y === target) return [i, j];
    }
  }
}
```

# 使用 hash table 优化

hash table 能够将已经遍历过的元素缓存起来，下次获取 diff 时间复杂度为 O(1)，这样省去了一次循环

```javascript
function twoSum(nums, target) {
  const cache = new Map();

  for (let i = 0; i < nums.length; i++) {
    const x = nums[i];
    const y = target - x;
    if (cache.has(y)) {
      return [i, cache.get(y)];
    } else {
      cache.set(x, i);
    }
  }
}
```
