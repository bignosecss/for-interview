全排列问题。
使用回溯算法暴力穷举所有可能的排序，将结果想象成一棵树；
要做的就是遍历这棵树，使用一个 track 记录从 根节点 -> 叶子结点的路径；
这个路径就是一种排列可能
```javascript
function permute(strs) {
  const result = [];
  backtrack(strs, [], result);
  return result;
}
function backtrack(strs, track, result) {
  if (track.length === strs.length) {
    result.push([...track]); // 这里要注意
    return;
  }

  for (const str of strs) {
    if (strs.includes(str)) continue;

    track.push(str);
    backtrack(strs, track, result);
    track.pop(str);
  }
}
```
