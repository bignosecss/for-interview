### 最简单的深拷贝：
```javascript
function simpleDeepClone(target) {
  return JSON.parse(JSON.stringify(target));
}
```
这种做法问题很大，`JSON.stringify()` 有其严格的序列化规则，无法处理：
1. undefined / Symbol / function
2. 无法处理循环引用
3. Date / RegExp 无法被保留其特性

### 基础版
递归的克隆对象的属性
```javascript
function basicDeepClone(target) {
  if (typeof target !== 'object' || target === nul) return target;

  let cloneTarget = Array.isArray(target) ? [] : {};
  for (const key in cloneTarget) {
    if (target.hasOwnProperty(key)) {
      cloneTarget[key] = basicDeepClone(target[key]);
    }
  }

  return cloneTarget;
}
```
问题：
1. 无法处理循环引用：无限递归，最终导致栈溢出
2. Date / RegExp

### 优化版
通过 `WeakMap` 解决循环引用
```javascript
function enhancedDeepClone(target, cache = new WeakMap()) {
  if (typeof target !== 'object' || target === null) return target;
  if (cache.has(target)) return cache.get(target);

  let cloneTarget = Array.isArray(target) ? [] : {};
  cache.set(target, cloneTarget);

  for (const key in target) {
    if (target.hasOwnProperty(key)) {
      cloneTarget[key] = enhancedDeepClone(target[key], cache);
    }
  }

  return cloneTarget;
}
```
问题：
1. Date / RegExp / Set / Map 等
